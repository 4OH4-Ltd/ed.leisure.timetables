About this timetable

This page shows Edinburgh Leisure swimming schedules in a cleaner, mobile-friendly timeline view.

  • Data is fetched from Edinburgh Leisure timetable APIs during GitHub Pages builds.
  • Refresh runs every 2 hours.
  • You can filter by venue and location, and click sessions for details.

Contributing

Contributions are welcome — whether that's new venue feed IDs, UI polish, or bug fixes.

  • Clone/fork the repo and create a feature branch.
  • Run npm install, npm run fetch:data, and npm run build.
  • Open a PR with a short summary and screenshots for visual changes.
